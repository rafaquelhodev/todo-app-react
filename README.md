# todo-app

Todo-app is a web app to manage your todo lists.

This app is powered by a restful API with mongoDB and built with React.js.

## Run docker container

```sh
docker-compose build
docker-compose up
```

## Run backend API

```sh
cd backend
npm run dev
```

## Run App

```sh
cd frontend
npm run dev
```

## Requirements
- Mongodb
- Node.js
